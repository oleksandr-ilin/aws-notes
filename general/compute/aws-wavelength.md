Overview
AWS Wavelength brings compute to the edge within telco networks for ultra-low-latency applications.

Tasks
- Deploy applications in Wavelength Zones, integrate with mobile network operators, and test latency.

Alternatives
| Alternative | Main pro | Main con |
|---|---:|---|
| Local Zones | Broader coverage | Possibly higher latency to mobile endpoints

Limitations
- Very limited geographic availability tied to carrier locations.

Price info
- Charged like regional compute plus data transfer specifics for carrier integrations.

Network & multi-region
- Zone-specific; plan integration with parent region for control and storage.

Popular use cases
- 5G mobile apps, AR/VR, and real-time gaming.

When Not to Use
- General web apps that do not require extremely low latency.

DR strategy
- Ensure fallback to parent region or other edge deployments for continuity.
