Overview
AWS Migration Hub provides a central place to track progress across multiple migration tools and projects.

Tasks
- Connect migration tools (DMS, MGN, Discovery) and monitor progress and status.

Alternatives
| Alternative | Main pro | Main con |
|---|---:|---|
| Spreadsheets / PM tools | Flexible | Manual and error-prone |
| Third-party trackers | Feature-rich | Integration work

Limitations
- Acts as a central tracker; most heavy lifting is done by integrated tools.

Price info
- Free to use for tracking; costs come from integrated services.

Network & multi-region
- Shows progress per-region as reported by tools; not a data mover.

Popular use cases
- Multi-tool migration programs needing consolidated visibility.

When Not to Use
- One-off small migrations where overhead isn't justified.

DR strategy
- Reuse Migration Hub for planned DR exercises and migration runbooks.
